# 👥 Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project groups customers into different segments based on their spending behavior and income using unsupervised machine learning.

It helps businesses understand customer types for better marketing strategies.

---

## 🎯 Objective
- Identify customer segments
- Analyze spending behavior
- Use clustering to group similar customers
- Visualize hidden patterns in data

---

## 📂 Dataset
- Mall Customers Dataset
- Features:
  - Customer ID
  - Gender
  - Age
  - Annual Income
  - Spending Score

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🤖 Machine Learning Techniques Used
- K-Means Clustering
- Elbow Method (to find optimal clusters)
- Hierarchical Clustering
- PCA (for visualization)
- Silhouette Score (cluster evaluation)

---

## 📊 Steps Performed
1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Visualization
4. Elbow Method for optimal K
5. K-Means Clustering
6. Cluster Visualization
7. PCA Visualization

---

## 📈 Key Insights
- Customers can be grouped into different segments:
  - High income high spenders
  - High income low spenders
  - Low income high spenders
  - Low income low spenders
- Businesses can target each segment differently

---

## 📊 Visualization
- Income vs Spending Score clusters
- PCA-based cluster visualization
